A smooth ajax-based contact form for informing web admin to contact back.

== Description ==

WTI Contact Back is a plugin for sending website admin an email letting him know that he needs to contact you back. Its simple by having only name and email fields to fill up. It comes with a language file for en-US(english- United States).

Some of the useful features:
* AJAX based contact processing.
* No need to fill up long form. Just fill up name, email and submit.

**Plugin URL:** http://www.webtechideas.com/wti-contact-back-plugin/

== Installation ==

* Download the plugin and extract it.
* Upload the directory '/wti-contact-back/' to the '/wp-content/plugins/' directory.
* Activate the plugin through the 'Plugins' menu in WordPress.
* Click on 'WTI Contact Back' link under Settings menu to access the admin section.
* On widgets section, there is a widget called 'WTI Contact Back' available which is used as the widget.
